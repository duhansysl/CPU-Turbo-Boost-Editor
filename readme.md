CPU Boost Advanced Control is a lightweight Windows command-line tool that allows you to switch CPU Turbo Boost behavior via the “Processor performance boost mode” power plan setting (requires Administrator privileges and does not modify voltage or perform overclocking).

Modes:

* Disabled – Completely disables Turbo Boost and locks the CPU to base frequency for lower temperatures and power consumption.
* Enabled – Standard balanced Turbo Boost behavior suitable for everyday use.
* Aggressive – Prioritizes maximum performance with faster and more responsive boosting.
* Efficient Enabled – Focuses on power efficiency and lower thermals with controlled boost behavior.
* Efficient Aggressive – Balanced mode combining strong performance with improved thermal stability.
* Show Current Boost Mode Value – Displays the currently active boost mode.
